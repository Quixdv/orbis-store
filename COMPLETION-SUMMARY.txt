## Installation on PS4

### Step 1: Build Package
```bash
cd ps4 && source /home/neox/.orbis-env.sh && make clean && make
# Result: OrbisStore.pkg created
```

### Step 2: Copy to USB
```bash
cp ps4/OrbisStore.pkg /mnt/usb/
```

### Step 3: Install on PS4
1. Insert USB into PS4
2. Settings → System Software → Update from USB
3. Wait 2-5 minutes for installation
4. App should appear in library

### Step 4: Test Launch
- Application should **NOT crash**
- **No CE-30008-1 error**
- UI should display normally
- Database should create at `/data/OrbisStore/`
